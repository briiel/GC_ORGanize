# GC_ORGanize

Frontend workspace for GC-ORGANIZE.

## Quick Start

1. `cd gc_organize`
2. `npm install`
3. `npm run start`
4. Open `http://localhost:4200`

## Build

- `npm run build`

## Key Docs

- `gc_organize/README.md`
- `gc_organize/DESIGN_SYSTEM.md`
- `gc_organize/MIGRATION_GUIDE.md`

## App Notes

- Role-based dashboards use routes under `/student-dashboard`, `/org-panel`, and `/osws-admin`.
- API base URL is configured in environment files under `gc_organize/src/environments`.
